    • Electron 介绍

    Electron是一种跨平台的桌面应用的解决方案，使用 JavaScript, HTML 和 CSS 构建跨平台的桌面应用。这就意味着你可以利用现有很多前段UI库做出很酷炫的应用。比如：

    1. Highcharts(图表库)
    2. echartsjs（也是一个图表库）

      可以来看看用 Electron制作的软件：

      可以这么说 Electron 的技术已经相当成熟了，可以干很多事情了。

    其他的我就不多说了，因为最初学习Electron是因为要做一个数据展示界面，自己也不是专业的程序员。这个文章也是自己学习的一个记录，如果有不对的地方还希望各位和我多多交流，互相学习，一起进步。这个课程很大程度上是借鉴了李宁老师的课程，更多的是一个记录，方便日后学习，也做一个抛砖引玉。

    一、课前的准备

    1. 你先要准备安装好node.js
    2. 同时在笔记本上面安装好git
    3. 安装好 electron
    在这里需要讲一下怎么 安装electron
    • 整确保你已经安装了 node.js
    • 在 cmd 中执行 npm install electron -g(在这里执行的是全局安装)
    • 在 cmd 中执行electron ，如果你看到了以下这个界面，就说明你已经成功了

    二、至少需要三个文件

    1. package.json: 用于配置Electron 工程的

    下面的 "main":"index.js" 是必要的，是整个程序的入口

    {
  "name": "first", 
  "version": "1.0.0",
  "main": "index.js"
}
    1. index.js: 主程序桌面应用程序入口点
    const {app,BrowserWindow} = require('electron'); // 在electron 中有 app 这个对象 和BrowserWindow 这个类

function createWindow(){  
  win = new BrowserWindow({width:800,height:600}); //在这里是创建一个BrowserWindow的实例，就是一个窗口，设定大小
  win.loadFile('index.html'); //加载了主窗口UI的页面文件
}

app.on('ready',createWindow); //app对象下面监听 'ready' 这个时间，只有当app准备好了才会启动窗口
    1. ==index.html==：主窗口UI的页面文件
      <!DOCTYPE html>
<html lang="en">
<head>
<meta charset="UTF-8">
<meta name="viewport" content="width=device-width, initial-scale=1.0">
<meta http-equiv="X-UA-Compatible" content="ie=edge">
<title>Hello World</title>
</head>
<body>
这是第一个electron 应用
</body>
</html>
